# Snake Game 🐍

A classic Snake game that runs entirely in the browser — no dependencies, no build step required.

## How to play

Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari) and click **Start Game**.

| Key | Action |
|-----|--------|
| ↑ / W | Move up |
| ↓ / S | Move down |
| ← / A | Move left |
| → / D | Move right |
| P | Pause / Resume |

## Features

- Smooth, grid-based movement
- Score tracking with high score saved in `localStorage`
- Levels — the snake speeds up as your score grows
- Clean dark-themed UI
- No build tools or dependencies required
